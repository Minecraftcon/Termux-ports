# **INSTALLATION**
1⟩ To install the deb file, download the deb and put it in a accessible location.
2⟩ Run 
```apt install x11-repo```
```apt install <path_to_deb>```
And yay, you shoud have retroarch installed
(its a gui pogram so make sure to run it in a x11 framebuffer)
